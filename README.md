# GPS Quest
*A simple app for creating, playing, and sharing GPS quests*

### Motivation
I am making this app for the Horizons event by Hack Club, so I can go to the hackathon.

### Quest format
The normal quest in this app works like this. You get a tip where you need to go - text or something else, and when you arrive at that spot, you get a new tip.

> **Inspiration** - There are so many location related quests out there, but not many with this type of quest, so I will try to make my own ))
