# Features
At the time, this app features only quest **playing**, **editing** and **saving** - _no_ sharing.

## Quest format
Each quest contains some tasks, where each task is a tip for a specific location you need to go next. After you arrive at this location, you get a new task.
As metadata each quest also has an id and a name

### Task structure
each task contains:
- name
- description
- coordinate
- radius

### Playing quests
When you playing a quest, the app shows the **quest progress**, and your **approximate distance** (encryted as a heat level).

One other cool feature is, thet you can **view older tasks**, just by scrolling to the left.

### Editing / creating quests
You can edit edit everything a quest includes, add new tasks reorder them or delete. Afterwards you can save it

_Editing is a bit buggy for now, but I am still working on it )_
